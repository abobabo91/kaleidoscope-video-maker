# Kaleidoscope Video Maker

Single-file in-browser tool that turns any image into an animated kaleidoscope and exports it as a WebM video.

**Live:** https://abobabo91.github.io/kaleidoscope-video-maker/

## Features
- Symmetry axes + horizontal/vertical mirror tiling
- Animated rotation, drift, and zoom (in-out or zoom-in-only)
- Pan/zoom the source image to pick the sample window
- Video export (720p / 1080p / 2K / 4K · 15–60 fps · 1–20 Mbps)

Pure client-side — no upload, no backend. Works in any modern Chromium/Firefox browser that supports `MediaRecorder`.
